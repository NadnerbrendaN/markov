# Markov Chains
A small C++ program to implement a markov-chain-based text predictor

When given a corpus of text, the program analyzes it and produces text output that mimics the style.
Edit "context_length" to make it have more/less context when writing.

## Parameters
{compiled name} -\[fd\] \[(filename)|(data)\]
- -f : file input
- -d : direct data input
